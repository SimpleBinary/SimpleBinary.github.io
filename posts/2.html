2. How to Use Icons in HTML

Hi, I'm SimpleBinary and today I'm going to teach you how to use icons in HTML.

Sometimes, you might go into a website and wonder how they have a Python logo that changes colour. This is possible with icons.

Icons are basically text, meaning that they can do anything that text can do, such as change colour, resize, or strikethrough.

Now that you know what icons are and what they can do, we will now learn How to use them. There are multiple ways to add logos to HTML code, I will list 3.


1. Font Awesome 5
Font awesome 5 has 1480 free icons and many more with font awesome pro. To use, simply put this link in the head section of your page. Eg.

<!DOCTYPE html>
<html>
  <head>
    <link 
          rel="stylesheet" 
          href="https://use.fontawesome.com/releases/v5.6.3/css/all.css" 
          integrity="sha384-HRtZLI+pbxtHCWp1t77Bi1L4ZtiqrqD80Kn4Z8NTSRyMA2Fd33n5dQ8lWUE00s/" 
          crossorigin="anonymous">
  </head>
  <body>
    <!-- After you have put the link in the head you can add icons -->
  </body>
</html>

Once you have added that link, go to Font Awesome's Website to learn how to use.


2. Bootstrap Icons
Bootstrap has lots of completely free icons that you can use if you put in the following link in the head section of your page.

<!DOCTYPE html>
<html>
  <head>
    <link 
rel="stylesheet" 
href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
  </head>
  <body>
    <!--Once you have added that link you can use bootstrap icons-->
  </body>
</html>

Learn more about Bootstrap icons here.


3. Google Icons
Google Icons has many free icons, to use put this line in your head element.

<!DOCTYPE html>
<html>
  <head>
    <link 
rel="stylesheet" href="https://fonts.googleapis.com/icon? 
family=Material+Icons">
  </head>
  <body>
    <!--Then you are done-->
  </body>
</html>

Find out more here.
